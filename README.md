# TdSnacks

#### 项目介绍
本项目开源但不允许商用，仅用于学习交流。

#### 软件架构
本网站使用纯JSP作为网站后台代码，对于SSH或SSM框架学习的开发人员没有多少借鉴意义，这样的大神请绕道。
本项目只是为了参加校内比赛，时间问题，有部分小bug，请读者根据说明提示，自行改正。

#### 安装教程

1. 克隆或下载本项目代码，并导入自己工作空间。
2. 还原数据库,我的数据库版本是sqlserver 2012
3. 修改数据的账号和密码,我的为 sa qwe1314521(Eclipse有项目内全局替换功能)
